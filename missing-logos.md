## How to Add Missing Logos

### Step 1: Obtain Logo Files
Contact each affiliate to request their logo files in the following preferred formats:
- **PNG** (preferred for logos with transparency)
- **JPG/JPEG** (for simple logos without transparency)
- **SVG** (vector format, ideal for scalability)
- **WebP** (modern format with good compression)

### Step 2: Logo File Requirements
- **Minimum size:** 200x200 pixels
- **Preferred size:** 400x400 pixels or larger
- **Format:** PNG with transparent background preferred
- **File naming:** Use lowercase, hyphens instead of spaces
- **Example:** `company-name-logo.png`

### Step 3: File Placement
1. Save logo files to: `/public/images/connections/`
2. Update the corresponding affiliate record in `/public/data/data.json`
3. Set the `logoUrl` field to: `/images/connections/filename.ext`

### Step 4: Update Data File
For affiliates with `"logoUrl": null`, update the entry:
```json
{
  "id": "affiliate-id",
  "title": "Affiliate Name",
  "organization": "Company Name",
  // ... other fields
  "logoUrl": "/images/connections/company-logo.png"
}
```

## Directory Structure

The logo files should be placed in the existing directory structure:
```
/public/
  /images/
    /connections/     <- Logo files go here
    /affiliates/      <- Other affiliate-related images
    [other image directories]
```
